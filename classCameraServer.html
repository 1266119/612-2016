Run the M-JPEG server.

This function listens for a connection from the dashboard in a background thread, then sends back the M-JPEG stream.

void CameraServer::SetImage ( Image const *  image)

Manually change the image that is served by the MJPEG stream.

This can be called to pass custom annotated images to the dashboard. Note that, for 640x480 video, this method could take between 40 and 50 milliseconds to complete.

This shouldn't be called if StartAutomaticCapture is called.

Parameters
imageThe IMAQ image to show on the dashboard

void CameraServer::StartAutomaticCapture ( char const *  cameraName = kDefaultCameraName)

Start automatically capturing images to send to the dashboard.

You should call this method to just see a camera feed on the dashboard without doing any vision processing on the roboRIO. SetImage shouldn't be called after this is called.

Parameters
cameraNameThe name of the camera interface (e.g. "cam1")
